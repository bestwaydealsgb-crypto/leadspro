# LeadHunter Pro

An AI-powered Lead Generation and Website Analysis System built with Next.js 14, TypeScript, Tailwind CSS, Supabase, and OpenAI.

## Features

### Lead Generation
- Search and find businesses in any niche/city
- Automatic website detection and analysis
- Deep website scoring (Mobile, SEO, Design, Speed)
- Contact information extraction (phones, emails, social media)

### Website Analysis
- Mobile responsiveness checking
- Page speed analysis
- SEO optimization scoring
- Design quality assessment
- Issues identification and reporting

### Lead Management
- Dashboard with real-time statistics
- Advanced filtering and search
- Lead prioritization
- Lead status tracking
- Export to Excel with color coding

### AI Integration
- Personalized outreach message generation
- Multi-channel messaging (WhatsApp, Email, Instagram, SMS)

## Getting Started

First, install dependencies and set up environment:

```bash
npm install
cp .env.example .env.local
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
